# Pokémon Typen-Trainer

Eine kleine lokale Lern-App für die Pokémon-Typentabelle.

## Starten

Öffne `index.html` im Browser.

## Als Android-App installieren

Die App ist als PWA vorbereitet. Damit Android sie installieren kann, muss sie über eine Web-Adresse laufen, am besten über HTTPS.

Gute Optionen:

- Für richtiges Installieren: den Ordner bei GitHub Pages, Netlify oder einem anderen HTTPS-Host veröffentlichen.
- Zum lokalen Testen am PC: im Ordner einen kleinen lokalen Server starten und `http://localhost:PORT` öffnen.

Danach in Chrome auf Android: Menü öffnen und `Zum Startbildschirm hinzufügen` bzw. `App installieren` wählen.

## Enthalten

- Kampf-Rechner: Angriffstyp gegen einen oder zwei Verteidigungstypen
- Defensivprofil: Schwächen, Resistenzen und Immunitäten eines eigenen Typs
- Pokémon-Beispiele mit Bildern für Ziel-Typkombinationen
- Merksätze für effektive, nicht effektive und immune Treffer
- Quiz mit Übungsmodus und zufälliger 50-Fragen-Runde
- 50-Fragen-Runde mit Endauswertung, Fehlerliste und Problemtypen
- Umschaltbarer Darkmode
- PWA-Dateien für Android-Installation und Offline-Cache
- Komplette Typentabelle für die moderne Gen-6+-Typenlogik
